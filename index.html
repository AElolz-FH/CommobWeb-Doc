Instructions pour lancer le projet¶

Configuration initiale¶

  1. Initialiser avec les pré-requis

    Consultez la documentation à cette adresse : Documentation CommobWeb

  2. Installer les ressources nécessaires au script Python

    pip install spacy flask flask-cors groq

  3. Télécharger le modèle SpaCy

    python -m spacy download fr_core_news_md

  4. Lancer le script Python

    python script.py

  5. Créer le dossier pour les images contextuelles et ajuster les permissions

    mkdir -p public/uploads/context_images
chmod -R 775 public/uploads/context_images

Une fois le serveur web lancé et le script Python exécuté, vous pouvez accéder à CommobWeb.

Déploiement avec Docker¶

  1. Build de l’image Docker

    Construisez l’image Docker à partir du fichier Dockerfile :

    docker build -t commobweb -f docker/Dockerfile .

  2. Se rendre dans le dossier Docker

    cd docker

  3. Lancer les conteneurs

    docker-compose up -d

  4. Accéder à l’application

    Ouvrez votre navigateur et accédez à : http://localhost:8080/

  5. Installer les ressources nécessaires au script Python

    pip install spacy flask flask-cors groq

  6. Télécharger le modèle SpaCy

    python -m spacy download fr_core_news_md

  7. Lancer le script Python

    python script.py

  8. Créer le dossier pour les images contextuelles et ajuster les permissions

    mkdir -p public/uploads/context_images
chmod -R 775 public/uploads/context_images

Une fois le serveur web lancé et le script Python exécuté, vous pouvez accéder à CommobWeb.
